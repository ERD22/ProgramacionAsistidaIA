# Ejercicios JavaScript

Colección de ejercicios prácticos para aprender y reforzar conceptos de JavaScript.

## 🚀 Características

- Ejercicios de fundamentos de JavaScript (variables, tipos de datos, operadores)
- Ejercicios de estructuras de control (condicionales, bucles)
- Ejercicios de funciones y arrow functions
- Ejercicios de arrays y objetos
- Ejercicios de DOM manipulación
- Ejercicios de ES6+ features

## 📋 Requisitos Previos

- Node.js instalado (opcional, para ejecutar ejercicios en consola)
- Navegador web moderno (Chrome, Firefox, Edge, Safari)
- Editor de código (VS Code, Sublime Text, etc.)

## 🔧 Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/tu-usuario/ejercicios-javascript.git
```

2. Navega al directorio del proyecto:
```bash
cd ejercicios-javascript
```

## 📖 Uso

Cada ejercicio está contenido en su propio archivo. Puedes ejecutarlos de las siguientes formas:

**En el navegador:**
1. Abre el archivo HTML correspondiente al ejercicio
2. Abre la consola del navegador (F12) para ver los resultados

**En la terminal con Node.js:**
```bash
node ejercicio-1.js
```

**Estructura del proyecto:**
```
├── basics/          # Ejercicios básicos
├── control-flow/    # Estructuras de control
├── functions/       # Funciones
├── arrays/          # Arrays
├── objects/         # Objetos
├── dom/             # Manipulación del DOM
└── es6/             # Características ES6+
```

## 🛠️ Desarrollo

Para añadir nuevos ejercicios:

1. Crea un nuevo archivo en la carpeta correspondiente
2. Sigue el formato de nombre: `ejercicio-descripcion.js`
3. Incluye comentarios explicativos
4. Añade el ejercicio a este README con una breve descripción
